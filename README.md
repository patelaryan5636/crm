# 🚀 Graphura CRM System

A secure, scalable, and performance-driven CRM (Customer Relationship Management) system built to streamline lead management, sales workflows, and organizational operations.

---

## 📌 Overview

Graphura CRM is designed to help businesses efficiently manage leads, track sales performance, and improve team collaboration. It provides a centralized platform with role-based access and real-time insights to optimize decision-making.

---

## ⚙️ Key Features

* 🔐 Role-Based Access Control (RBAC)
* 📊 Lead Management (Create, Update, Track)
* 📂 Bulk Lead Upload (CSV/Excel with validation)
* 📈 Sales Performance Tracking
* 🧾 Payment Status Monitoring (Success/Failed Alerts)
* 📍 Login Tracking (IP, Time, Location)
* 📱 Mobile App Integration (for sales teams)
* 🔔 Notifications & Alerts System
